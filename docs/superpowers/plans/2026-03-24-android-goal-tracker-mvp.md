# 안드로이드 골 트래커 MVP 구현 계획

> **에이전트 작업용:** 이 계획을 실행할 때는 `superpowers:subagent-driven-development` 또는 `superpowers:executing-plans`를 사용한다. 단계 추적은 체크박스(`- [ ]`) 형식을 기준으로 한다.

**목표:** 여러 프로젝트 추적, 레거시 저장, 응원 문구, 포커스 프로젝트 위젯 1종을 포함한 실제 설치 가능한 안드로이드 APK를 만든다.

**구조:** Kotlin + Jetpack Compose 기반 단일 앱으로 간다. 데이터는 Room에 로컬 저장하고, ViewModel과 Repository에서 상태를 묶는다. 위젯은 동일한 포커스 프로젝트 상태를 사용한다. 오늘 안에 끝내야 하므로 화면 구조와 내비게이션은 얕게 유지한다.

**기술 스택:** Kotlin, Jetpack Compose, Material 3, Navigation Compose, Room, Android App Widget, Gradle Wrapper, JUnit
